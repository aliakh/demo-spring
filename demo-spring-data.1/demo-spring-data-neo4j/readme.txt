1) Open browser at https://neo4j.com/download-center/
2) Download latest version of Neo4j Community Server for Windows in zip archive
3) Unzip the archive to D:\neo4j
4) Run Neo4j server as console application: D:\neo4j\bin\neo4j console
5) Open browser at http://localhost:7474
6) Use default login is username 'neo4j' and password 'neo4j'
7) Change password to 'secr3t'
